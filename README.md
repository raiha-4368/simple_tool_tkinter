# Simple Tool Tkinter
## Tkinterを使用したシンプルなツール
Tkinterを使用したシンプルなツール達をまとめたリポジトリ

## 使用技術/環境/起動及び使用手順
- それぞれのディレクトリ下にそれぞれ記載  

## Tools
 - contdown  
 時間を設定し、0になるまで時間を計測するツール    

- counter  
 数をカウントするシンプルなツール  

 - csv_viewer  
 csvファイルを開き、その内容を表示する(編集不可)    

- extension_sort  
 ディレクトリを選択し、その直下にあるファイルの拡張子ごとにフォルダに振り分けるツール   

 - memo  
 テキストエリアに記述した内容を保存する/テキストファイルを開きその内容を表示、  
 変更内容を上書き保存できるツール    

 - pomodoro_timer  
 ポモドーロのサイクル(25分作業5分休憩を4回繰り返す)を計ることができるツール    

 - rename_files  
 ディレクトリを選択し、その直下にあるファイル名を一括で変更することができるツール    

- serial_number_files  
 ディレクトリを選択し、その直下にあるフォルダ及びファイルに連番を付与することができるツール   

- stopwatch  
 start/stop/resetができるストップウォッチ

# exe化方法(各フォルダ直下で実行)
■pyinstallerのインストール  
以下コマンドを実行  
pip install pyinstaller   

exe化の方法  
1.以下コマンドを実行  
pyinstaller main.py --onefile  --noconsole --icon=icon_01.ico

使用しそうなオプション  
--onefileは1つのファイルにまとめる  
--noconsoleはコンソールを表示しない  
--icon=test.icoはアイコンを変更(*.iconファイルを同一ディレクトリに配置する)  

2.作業ディレクトリ内にbuildディレクトリ/distディレクトリが作成される  
distディレクトリ内にあるmain.exeをダブルクリックで実行出来るようになっている(筈)  

3.エラーになった場合は  
コンソールからmain.exeを実行し、エラーを確認する(--noconsoleを設定していると出ない可能性あるので注意)

youtubeなどの利用者(視聴者/クリエイター)に向けたツール作成  
以下Gemini解答
1. 視聴者・リサーチ向けツール
YouTubeの膨大なデータを活用して、ユーザーが動画を探したり理解したりするのを助けるツールです。
動画の要約・インデックス作成ツール
内容: 長い動画の文字起こしを取得し、AI（ChatGPTなど）で要約したり、重要なシーンのタイムスタンプを自動生成したりします。
使うライブラリ: youtube-transcript-api（字幕取得）、openai（要約）。
コメント分析ダッシュボード
内容: 特定の動画のコメントを数千件取得し、ポジティブ・ネガティブの割合をグラフ化したり、よく使われる単語（ワードクラウド）を表示したりします。
使うライブラリ: google-api-python-client（公式API）、pandas（データ処理）、Streamlit（画面作成）。 

2. クリエイター向け自動化ツール
動画投稿者が面倒に感じる作業を自動化するツールです。
ショート動画自動生成ツール
内容: 横長の動画から面白いシーンを切り抜き、縦型（9:16）にリサイズして字幕を付ける作業をプログラムで行います。
使うライブラリ: moviepy（動画編集）、yt-dlp（動画取得）。
トレンド・キーワード調査ツール
内容: 特定のジャンルで今伸びている動画のタイトルやタグを一覧にし、次に作るべき企画を提案します。
使うライブラリ: pytrends（Googleトレンド連携）、google-api-python-client。

3. ダウンロード・管理系ツール
自分のお気に入りのコンテンツを整理するためのツールです。
プレイリスト一括バックアップツール
内容: 自分の「後で見る」や特定のプレイリストに入っている動画を、最高画質で一括保存します。
使うライブラリ: yt-dlp（非常に強力なダウンロードライブラリ）。
特定チャンネルの更新通知ボット
内容: お気に入りのチャンネルが更新されたら、DiscordやLINEに通知を飛ばします。
使うライブラリ: requests（通知送信用）、公式API。
