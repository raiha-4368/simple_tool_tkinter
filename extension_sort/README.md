# ファイル拡張子別振り分けアプリ
## tkinterを使用した拡張子別振り分けツール
拡張子ごとに自動でファイルを仕分けするアプリケーション
※拡張子がないファイルはnot_extフォルダに格納されます。

## できること
- フォルダの選択/フォルダ場にあるファイルを表示  
- フォルダ内のファイルを拡張子の名前のフォルダを作成し、それらを拡張子ごとに振り分ける
  また拡張子が無いファイルの場合not_extフォルダを作成し、そこに格納する

## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(対象ディレクトリ下)で以下コマンドを実行
python main.py

1. "フォルダ選択"ボタンを押下  
2. ダイアログでフォルダを選択。左側のリストボックスにそのフォルダ下にあるファイルの一覧が表示されます。  
3. "プレビュー"ボタンを押下し、右側のリストボックスに振り分け後のファイルパスが表示されます。  
 ※この時点ではプレビューです。実際の変更は行われていません。  
 ※プレビューは必須動作です。プレビューを実行していない場合、振り分け処理は行われません。  
4. "拡張子別振り分け実行"ボタンを押下することで、プレビューのパス通りにディレクトリの作成及びファイルの振り分けを行います。  
※振り分けの際、対象ディレクトリに同一名称のファイルがあった場合、それらのファイルの振り分け実行を行わず、出来るファイルのみ振り分けます。

## 簡易テスト
### ■正常系
- フォルダ選択 → プレビュー → 実行 → 成功ダイアログが表示され、正常に振り分けされる  
- フォルダ選択 → クリア → 初期状態に戻る  

### ■異常系
- フォルダ未選択でプレビュー → エラーダイアログ表示  
- フォルダ未選択で振り分け実行 → 警告ダイアログ表示

### ■境界・特殊ケース
- 拡張子無しファイル → not_extフォルダに入る  
- フォルダが混在 → フォルダに対しては何もしない  
- 実行後に再度実行 → 警告ダイアログが表示
