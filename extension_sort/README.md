# フォルダ内連番付与アプリ
## tkinterを使用したフォルダ内連番付与ツール
拡張子ごとに自動でファイルを仕分けするアプリケーション
※拡張子がないファイルはnot_extフォルダに格納されます。

## できること
- フォルダの選択/フォルダ場にあるファイルを表示  
- フォルダ内のフォルダ及びファイルの先頭に連番を付与する/そのプレビュー機能  


## 使用技術
- Python
- Tkinter

## 環境
- Python 3.10 以上
- Windows

## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(対象ディレクトリ下)で以下コマンドを実行
python main.py

1. "フォルダ選択"ボタンを押下  
2. ダイアログでフォルダを選択。左側のリストボックスにそのフォルダ下にあるファイルの一覧が表示されます。  
3. "連番開始数(数値のみ)"に連番開始の数値を入力(デフォルトは1)    
4. "桁数設定"ラジオボタンで任意の桁数を設定(デフォルトは2)  
 例:  
	1の場合:1  
	2の場合:01  
	3の場合:001  
5. "プレビュー"ボタンを押下し、右側のリストボックスに変更後のディレクトリのファイルの一覧が表示されます。  
 ※この時点ではプレビューです。実際の変更は行われていません。  
 ※プレビューは必須動作です。プレビューを実行していない場合、連番付与処理は行われません。
6. "連番付与実行"ボタンを押下することで、フォルダ内のフォルダ及びファイルに連番付与処理が実行されます。

## 簡易テスト
### ■正常系
- フォルダ選択 → プレビュー → 実行 → 成功ダイアログが表示され、正常に振り分けされる  
- フォルダ選択 → クリア → 初期状態に戻る  

### ■異常系
- フォルダ未選択でプレビュー → エラーダイアログ表示  
- フォルダ未選択で振り分け実行 → 警告ダイアログ表示

### ■境界・特殊ケース
- 拡張子無しファイル → not_extフォルダに入る  
- フォルダが混在 → フォルダに対しては何もしない  
- 実行後に再度実行 → 警告ダイアログが表示
