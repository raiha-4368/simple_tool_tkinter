# 特定文字列抽出アプリ
## tkinterを使用した文字列抽出ツール
特定文字列を含む行(その行の先頭から改行/EOFまで)をファイルorフォルダから抽出するアプリケーション

## できること
- ファイルを選択し、そのファイルの記述内容を表示する  
- 選択したファイル内から入力した文字列を検索し、その文字列を含む行を抽出・表示する
- 検索結果をファイルにcsv形式でファイルへ出力する  

## 使用技術
- Python
- Tkinter

## 環境
- Python 3.10 以上
- Windows

## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(対象ディレクトリ下)で以下コマンドを実行
python main.py

1. "ファイル選択"ボタンを押下  
2. ダイアログでファイルを選択。左側のテキストエリア(編集不可)に表示されます。  
3. 検索文字列入力欄にファイル内から検索したい文字列を入力。  
デフォルトは部分一致、大文字小文字を区別する設定です。
完全一致及び大文字小文字を区別したい場合は、それぞれのチェックボックスにチェックを入れてください。
4. "検索開始"ボタンを押下し、右側のテキストエリア(編集不可)に検索対象が表示されます。  
5. 必要に応じて"ファイルへ出力"ボタンを押下することで任意の場所に検索結果をcsv形式で保存できます。

## CSV形式の出力ファイル
column行:ファイルパス,記述されていた行数,対象行の記述内容
デフォルトのファイル名:{検索文字列}_YYYYMMDD_hhmmss.csv
例:
test_20260401_174506.csv  
filepath,line_num,content  
D:/git/{filepath}/text_file2.txt,1,test  aaa  
D:/git/{filepath}/text_file2.txt,2,test2 bbb

## 簡易テスト
### ■正常系
- ファイル選択 → 検索開始 → ファイルへ出力(部分一致) → 成功ダイアログが表示され、csvファイルに出力される  
- ファイル選択 → 左側のテキストエリアにファイル内容が表示 → クリア → 初期状態に戻る  

### ■異常系
- ファイル未選択で検索開始 → エラーダイアログ表示  
- ファイル未選択でファイルへ出力 → 警告ダイアログ表示  
- ファイルを選択し、検索文字列を入力せず検索開始 → エラーダイアログ表示  

### ■境界・特殊ケース
- 検索開始(完全一致) → 完全一致のみの抽出  
- 検索開始(大文字小文字を無視) → 大文字小文字を無視して抽出  
- 検索開始(完全一致&大文字小文字を無視) → 完全一致&大文字小文字を無視して抽出  
- 複数行の検索対象あり → 複数行の抽出
