## できること
- フォルダの選択/フォルダ場にあるファイルを表示  
- ファイル中にある文字を任意の文字に置換する/そのプレビュー機能  


## 使用技術
- Python
- Tkinter

## 環境
- Python 3.10 以上
- Windows

## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(対象ディレクトリ下)で以下コマンドを実行
python main.py

1. "フォルダ選択"ボタンを押下  
2. ダイアログでフォルダを選択。左側のリストボックスにそのフォルダ下にあるファイルの一覧が表示されます。  
3. "変更前の文字列入力欄"に変更したいファイルの名称を入力  
4. "変更後の文字列入力欄"に変更後のファイルの名称を入力
 ※3.で指定した文字を4.の文字列に置換します。  
5. "プレビュー"ボタンを押下し、右側のリストボックスに変更後のディレクトリのファイルの一覧が表示されます。  
 ※この時点ではプレビューです。実際の変更は行われていません。  
 ※プレビューは必須動作です。プレビューを実行していない場合、ファイル名の変換処理は行われません。
6. "ファイル名一括変換実行"ボタンを押下することで、ファイル名の変換処理が実行されます。
